# MHStories-like-BattleSystem
 공부용으로 가위바위보 앱을 몬헌스토리즈 배틀시스템처럼 만들기   
 파워 > 스피드 > 테크니컬 > 파워 상성대로 배틀 결과가 나옴   
 이기면 상대에게 10 데미지를 주고 지면 10 데지미를 받고 비기면 아무런 데미지를 안받는다.   
 양쪽에 100 HP를 시작을하고 상대의 HP를 0으로 만들면 이긴다.
